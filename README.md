# Hollywood Bowls — Simple 3-page site

Files:

- [index.html](index.html) — Homepage
- [menu.html](menu.html) — Menu
- [contact.html](contact.html) — Contact form
- [css/styles.css](css/styles.css) — Styles

To view locally, open `index.html` in a browser or run a simple HTTP server:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
